What is Pashatoku?

Pashatoku is a fast-paced, interactive quiz platform where participants scan QR codes around a venue to answer questions in real-time.

Whether for school events, corporate team-building, or hackathons, it provides the open-source toolkit to host your own engaging quiz event without relying on third-party cloud services. You retain full control over your data, UI, and deployment.

📷 QR Code Integration

Participants can simply scan a QR code at physical locations to instantly open a question. No app installation required.

⏱️ Real-time Synchronization

Features live timers, global forced-end events, and real-time dashboard updates via modern web technologies.

🛠️ Fully Open Source

Built with Go (Backend) and React/Vite (Frontend). You can customize everything to fit your hackathon or classroom.

How to Use

For Organizers

1. Setup Server

Clone the repository and run `docker compose up -d` to start the PostgreSQL databases, Go backend, and React frontend.

2. Create Events

Access the Admin Dashboard to create quiz events, add questions, set time limits, and generate QR codes for each question.

3. Manage Live

Change quiz statuses (Waiting -> Started -> Ended), monitor participant scores, or trigger a global countdown timer.

For Participants

1. Register

Access the main URL, enter your name and student/employee ID to join the system.

2. Scan & Answer

Scan the event's QR codes using your device's camera to load specific questions. Submit answers to earn points.

3. View History

Track your progress and total score across different quiz events from the 'History / Results' tab.

Help & Troubleshooting

Q. The camera won't open for scanning QR codes.

A. Make sure you have granted camera permissions to your browser. Also, note that most mobile browsers require the site to be served over HTTPS (or localhost) for camera access to work.

Q. Why do I see a "GPS Verification Failed" error?

A. If the organizer enabled GPS restrictions for a question, you must be physically near the designated coordinates. Ensure Location Services are turned on for your browser.

Q. I answered a question but it says "Event Closed".

A. The organizer has manually closed the event, or the global timer has expired. Answers can no longer be submitted.

Q. Can I use Pashatoku as a cloud service?

A. No, Pashatoku is self-hosted software. You must deploy it on your own server or local network.
